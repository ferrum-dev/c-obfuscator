### 💥 АВТОМАТИЧЕСКОЕ ОПРЕДЕЛЕНИЕ ТИПА

**Определяет тип исполняемого файла и применяет оптимальную обфускацию:**

#### C# (.NET) Обфускация
- Обфускация .NET метаданных
- Шифрование строк в IL коде
- Anti-debug проверки
- Обфускация потока управления

#### C++ Обфускация
- Обфускация символов
- Шифрование строк в памяти
- Anti-debug паттерны
- Внедрение мусорного кода

### Anti-Debug
Автоматическое определение отладчика. Приложение завершится если обнаружен debugger.

### String Encryption
Все строки зашифрованы и расшифровываются во время выполнения.

### Symbol Obfuscation
Переименование всех символов в мусорные имена.

### Junk Code Injection
Добавление фейкового кода для усложнения анализа.
